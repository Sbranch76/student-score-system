# 学生成绩管理系统
信息系统实践课程项目

## 项目简介
本项目是一套基于 Spring Boot + MySQL 的轻量化学生成绩管理系统，实现登录验证、学生信息管理、成绩管理与成绩查询功能，满足课程设计与教学管理基本需求。

## 技术栈
- 后端：Spring Boot 3.x、Spring Data JPA、Lombok
- 前端：HTML5、CSS3、JavaScript、Bootstrap 5
- 数据库：MySQL 8.0
- 版本控制：Git + GitHub

## 项目结构
- backend/：后端代码目录
- frontend/：前端页面目录
- sql/：数据库建表脚本
- docs/：项目文档（计划书、测试文档、设计文档）
